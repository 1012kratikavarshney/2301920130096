# 🔔 Notification Management System

A modern web application that helps students efficiently manage and prioritize campus notifications such as Placement, Results, and Events.

## 📌 Project Overview

The Notification Management System is designed to organize notifications based on their importance and recency. It provides a clean and responsive interface where users can view, search, filter, and prioritize notifications.

This project was built as part of a frontend evaluation using HTML, CSS, JavaScript (or React), and REST API integration.

---

## 🚀 Features

- 📢 View all notifications
- ⭐ Priority Inbox (Placement > Result > Event)
- 🔍 Search notifications
- 🏷️ Filter notifications by type
- 📅 Sort by latest notifications
- 📱 Fully responsive design
- 🎨 Modern UI
- ⚡ Fast loading interface
- 🔄 REST API integration
- 📊 Dashboard with notification statistics

## 🔄 Workflow

1. Fetch notifications from the API.
2. Categorize notifications.
3. Assign priority based on notification type.
4. Sort by timestamp.
5. Display the Top Priority notifications.
6. Allow searching and filtering.
